# 🍎 Apple-Style 3D Website

An immersive Apple-style 3D website built with React, Three.js, GSAP, and Tailwind CSS. This project delivers cinematic scroll-driven animations, interactive 3D product scenes, smooth timeline transitions, and a fully responsive modern UI experience.

---

# ✨ Introduction

Apple-style 3D website built with React, Three.js, GSAP, and TailwindCSS!

Showcase products in immersive 3D scenes, scroll-animated models, and pinned sections. Featuring responsive design, smooth timeline animations, and visually striking image transitions—perfect for developers creating interactive, modern web experiences.

- Seamless timeline animations
- Custom animation sequences

# 🔋 Features

### 👉 3D Product Scene with Realistic Lighting System
Showcase products inside immersive 3D environments with cinematic lighting and realistic rendering.

### 👉 Animate 3D Model on Scroll
Create engaging user interactions by animating 3D models based on scroll position.

### 👉 ScrollTrigger Effects
Use GSAP ScrollTrigger to control animations and synchronize visual storytelling with scrolling.

### 👉 Pinned Sections
Lock sections in place during animation sequences for immersive storytelling experiences.

### 👉 Image Masking Effects
Create visually striking transitions using masks and scroll-based image reveals.

### 👉 Seamless Timeline Animations
Build smooth timeline animations that connect multiple sections into one continuous experience.

### 👉 Responsive Design
Ensure adaptive layouts and optimized animations across desktops, tablets, and mobile devices.
